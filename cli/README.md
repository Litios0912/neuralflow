# NeuralFlow CLI

CLI tool for the NeuralFlow AI Automation Platform.

## Installation

```bash
pip install -e .
```

## Usage

```bash
# Login
neuralflow login

# Register
neuralflow register

# List available agent types
neuralflow agent types

# List your agents
neuralflow agent list

# Create an agent
neuralflow agent create

# Run an agent
neuralflow agent run 1 "Your input here"

# Check API health
neuralflow health
```
